Data attributes of the relative distance measurement taken at a single timestep in Robotx\+\_\+\+Measurements. 

\begin{DoxyNote}{Note}
This structure combines multiple measurements with the same time stamp. There it is assumed that the vectors\+: subjects, ranges, and bearings are in the same order.
