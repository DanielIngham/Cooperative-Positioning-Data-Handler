\doxysubsection{Detailed Description}
Data attributes for a single groundtruth reading extracted from Robotx\+\_\+\+Groundtruth.\+dat. 

\begin{DoxyNote}{Note}
According the UTIAS website, the groundtruth readings are accurate to the order of 1mm (1E\texorpdfstring{$^\wedge$}{\string^}-\/3 m). Additionally, the NTP deamon used to synchronise the clocks between robots has an average timing error of 1ms (1E-\/3 s). 
\end{DoxyNote}
