Detailed Description

Data attributes for a single groundtruth reading extracted from Robotx_Groundtruth.dat.

Note
According the UTIAS website, the groundtruth readings are accurate to the order of 1mm (1E^-3 m). Additionally, the NTP deamon used to synchronise the clocks between robots has an average timing error of 1ms (1E-3 seconds).
